<span style="color:#55FF55">🚀 Как установить?</span>
1. Скачай .jar из Releases.
2. Кинь в папку `plugins`.
3. **Перезапусти сервер** (не `/reload`).
4. Настрой `config.yml` если нужно.
5. Готово.

> 💡 `/reload` ломает плагины — только рестарт.
