## 🚀 Как установить плагины?

1. Скачайте последнюю версию плагина из раздела **Releases** этого репозитория.
2. Поместите `.jar`‑файл в папку `plugins` вашего сервера Minecraft.
3. Перезапустите сервер или выполните команду `/reload`.
4. Настройте плагин через конфигурационный файл (если требуется).
5. Готово! Используйте новые возможности в игре.

> 💡 **Совет:** Перед установкой убедитесь, что ваш сервер соответствует требованиям плагина (версия ядра, зависимости).
